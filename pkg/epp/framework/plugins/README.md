# Plugins

This directory contains all built-in plugins for the llm-d-inference-scheduler, organized by subsystem.

## Subsystems

- [datalayer/](datalayer/) — data sources, extractors, and endpoint attributes
- [flowcontrol/](flowcontrol/) — admission, ordering, fairness, and saturation detection
- [requestcontrol/](requestcontrol/) — request-level data producers and admitters
- [requesthandling/](requesthandling/) — protocol parsers
- [scheduling/](scheduling/) — filters, scorers, pickers, and profile handlers
